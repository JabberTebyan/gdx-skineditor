gdx-skineditor
==============

A skin editor for libgdx 0.3

Easy way to make it run (from console):

```sh
git clone https://github.com/cobolfoo/gdx-skineditor.git skineditor
cd skineditor
./gradlew run
```

*** You could also use the distribution zip file from DIST folder. ***
